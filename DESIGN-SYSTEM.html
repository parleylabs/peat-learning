Adopt the design system, not the CDN.

The Visual Explainer skill is a strong quality bar. We take its components, tables, accessibility, and "no generic aesthetic" discipline — and bind it to our two hard constraints: everything stays self-contained & air-gap-safe, and everything wears the Parley Labs brand. No runtime Mermaid/Chart.js; pre-render to inline SVG.

⚑ The one rule that overrides the skill

The skill loads Mermaid & Chart.js from a CDN at runtime. Our pages must render offline — the Off the Grid track is about denied networks, and these pages may be opened air-gapped. So: diagrams are authored as ```mermaid in the markdown (GitHub renders them) and pre-rendered to inline SVG for the HTML; all CSS/JS is inline; the only external call remains the cookieless GoatCounter beacon. A CDN <script src> is a gate failure.

What we already have vs. what this adds

We're not starting over — roughly 70% of this is already shipped in the curriculum. The skill mostly formalizes and levels it up.

Already in place ✓

Self-contained pages · Parley Labs palette + Montserrat/Roboto · the Shipped/In-flight/Proposed/Speculative status palette · inline-SVG diagrams with legends · the diagram registry (code-derived, re-verified each sweep) · the analytics/feedback carve-out.

What the skill adds

A formal component library (cards, KPI, data-table) · accessibility (semantic HTML, ARIA, prefers-reduced-motion) · light-mode (we're dark-only today) · diagram zoom/pan/expand (inline JS) · a written quality checklist the gates can enforce.

How it becomes a weekly / monthly rule

1 · One written standard

Commit learning/DESIGN-SYSTEM.md (this page, distilled) as the canonical spec, referenced from CLAUDE.md.

2 · A visual gate, every refresh

Extend the diagram gate into a visual & design gate: self-contained (no CDN), semantic, reduced-motion, status palette, table discipline, no overflow, legend on every diagram, hierarchy-first.

3 · A monthly upgrade pass

The full sweep migrates a bounded batch of weak visuals (ASCII → SVG, dense prose → cards/tables) toward the standard — incremental, never big-bang.

4 · Improves via the ratchet

curriculum-feedback on confusing visuals + the §9b retrospective file prompt-amendment issues — the standard tightens with your approval.
